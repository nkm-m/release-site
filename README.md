# release-site

## Prerequisite
- Node.js
- AWS CLI
- AWS Profile
- Amplify CLI
## CLI Setup
1. `npm install`
2. `amplify init`
3. `amplify add auth`
4. `amplify add api`
5. `amplify update api`
6. `amplify add hosting`
7. `amplify publish`

## main.js Setup
1. input Project Name
2. input API Name
3. input Slack Incoming WebHook URL
